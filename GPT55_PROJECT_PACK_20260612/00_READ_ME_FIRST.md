1. Upload file zip cua goi nay len ChatGPT web.
2. Mo file `01_GPT55_MASTER_PROMPT.md`.
3. Copy noi dung trong file do va gui cho GPT.
4. Yeu cau GPT doc `02_PROJECT_BRIEF_SMART_HOME_AI.md` truoc khi lap ke hoach bai bao.
