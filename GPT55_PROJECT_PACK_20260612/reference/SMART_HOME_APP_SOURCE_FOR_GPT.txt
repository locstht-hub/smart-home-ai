- Dien thoai that: `http://<IP-LAN-Home-Assistant>:8123`

API Home Assistant dang dung:
- `GET /api/states`
- `POST /api/services/<domain>/turn_on`
- `POST /api/services/<domain>/turn_off`
- `POST /api/conversation/process`
- WebSocket event `state_changed` de dong bo trang thai/log.

DataContext hien tai:
- Load cache local tu AsyncStorage.
- Neu Home Assistant configured thi `refresh()` goi `client.getStates()`.
- Map states tu HA sang danh sach phong/thiet bi trong app.
- Toggle device se goi service `turn_on`/`turn_off` neu device co `entityId`.
- Neu khong co HA thi toggle local.
- Activity logs luu toi da 100 log moi nhat.
- Poll refresh moi 15 giay khi co HA.

4. PLC - HOME ASSISTANT - APP MAPPING

1. PLC/HMI cap nhat status tag.

3. App doc Home Assistant entity de hien thi ON/OFF.
4. Neu app gui lenh, Home Assistant/gateway ghi command tag.
5. PLC xu ly lenh xong cap nhat lai status tag.

Mapping mau hien co:
1. Den chinh phong khach: M100.0 / M110.0 -> light.living_main_light
2. Den hat phong khach: M100.1 / M110.1 -> light.living_ambient_light

4. May lanh phong ngu: M100.3 / M110.3 -> climate.bedroom_ac
5. Den ngu: M100.4 / M110.4 -> light.bedroom_light
6. Den bep: M100.5 / M110.5 -> light.kitchen_light
7. Quat hut bep: M100.6 / M110.6 -> fan.kitchen_exhaust_fan
8. O cam tu lanh: M100.7 / M110.7 -> switch.kitchen_fridge_outlet
9. Den garage: M101.0 / M111.0 -> light.garage_light
10. Cua cuon garage: M101.1 / M111.1 -> switch.garage_door_motor
