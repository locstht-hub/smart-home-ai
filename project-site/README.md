# Smart Home AI Project Site

Website tĩnh giới thiệu đồ án Smart Home AI.

## Chạy local

Mở trực tiếp:

```text
project-site/index.html
```

Hoặc dùng extension Live Server trong VS Code.

## Deploy Cloudflare Pages

Cloudflare Pages đang lấy source từ GitHub. Khi commit và push lên nhánh được cấu hình deploy,
Cloudflare sẽ tự build/deploy lại site.

Custom domain:

```text
smarthomeai.id.vn
www.smarthomeai.id.vn
```

API backend được tách riêng ở:

```text
api.smarthomeai.id.vn
```

## Cập nhật nội dung

- Sửa nội dung chatbot/FAQ trong `knowledge.js`.
- Sửa bố cục trong `index.html`.
- Sửa giao diện trong `styles.css`.
